## Fake News Detection System
A system designed to detect fake news by analyzing and classifying news articles.
It uses data-driven techniques to identify patterns associated with misinformation.
The project focuses on providing accurate and efficient news verification.

##  Features
* Classifies news articles as **Real or Fake**
* Processes textual data for prediction
* Supports multiple machine learning models
* Provides quick and reliable results
* Simple and user-friendly interface

##  Technologies Used
* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib

##  How to Run
bash
# Clone the repository
git clone https://github.com/Rithika2612/FakeNewsDetectionSystem.git

# Navigate to project folder
cd FakeNewsDetectionSystem

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py


##  How It Works
1. Load and prepare news data
2. Convert text into numerical format
3. Train machine learning models
4. Evaluate model performance
5. Predict whether news is real or fake


## Dataset
The dataset is not included in this repository due to size limitations.
You can use publicly available fake news datasets (e.g., Kaggle datasets) to run the project.
